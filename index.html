QUESTION 1. 

You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string.

SOLUTION. 
      //unique elements
      //insertion
      let s="abcadeecfb";
      //set stores unique elements only
      let uniqueCharacters=new Set();
      for(let i=0;i < s.length;i++)
        {
          uniqueCharacters.add(s[i]);
        }
      let result="";
      for(let key of uniqueCharacters.keys())
        {
          result+=key;
        }
      console.log(result);
Input 1: abcadeecfb
Output:  abcdef
